# Mixing colours in CSS

`color-mix` lets you mix colours in CSS at runtime, in any of the supported colour spaces.

```css
color-mix(in hsl, hsl(200 50 80) 20%, coral 80%)
```

Unlike [[20250729123817-css-relative-colors]], this seems to mix two colours explicitly. Rather than operate on a origin colour in various ways.

https://developer.chrome.com/docs/css-ui/css-color-mix
