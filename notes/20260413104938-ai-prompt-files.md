# Prompt files

Prompt files are a pretty simple concept that lets you simplify prompting for common tasks by encoding them as a standalone markdown file. Effectively like a board you can copy-paste from.

They should only be used for lightweight, single-task prompts.
If you need something more complex or portable, use [[20260306121850-ai-agent-skills]].
If you need a persistent persona, use [[20260306123345-ai-custom-agents]].
