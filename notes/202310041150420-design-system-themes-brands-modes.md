# Design system themes: brands vs modes

'Themes' in a design system is a bit of a misnomer. It can be helpful to segment themes into two different types:

Brands
	Static
	Mutually exclusive
	Build-time
	Hierarchical

Modes
	Dynamic
	Contextual
	Run-time
	Flat(?)
